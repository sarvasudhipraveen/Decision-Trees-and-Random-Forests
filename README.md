# Decision-Trees-and-Random-Forests
Decision Trees and Random Forests
Overview

This project demonstrates the implementation of Decision Tree and Random Forest classification algorithms using the Breast Cancer dataset. The objective is to compare the performance of both models and understand the power of ensemble learning.

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Steps Performed
Loaded the Breast Cancer dataset.
Performed train-test splitting.
Applied feature scaling.
Trained a Decision Tree classifier.
Visualized the Decision Tree structure.
Evaluated model performance using Accuracy and Classification Report.
Trained a Random Forest classifier.
Compared Decision Tree and Random Forest performance.
Visualized feature importance scores.
Results
Random Forest achieved better accuracy than Decision Tree.
Ensemble learning reduced overfitting.
Important features influencing classification were identified.
Learning Outcomes
Understanding of Decision Trees.
Understanding of Random Forests.
Knowledge of Bagging and Ensemble Learning.
Experience with feature importance analysis.
Practical exposure to classification model evaluation.
